# 💰 Budget Tracker

Dit is een eenvoudige budgettracker die ik heb gemaakt in Python. Met dit programma kun je je inkomsten en uitgaven bijhouden, je saldo bekijken en alles automatisch opslaan in een JSON-bestand.

Dit project heb ik gemaakt om meer te oefenen met Python, zoals werken met bestanden, lussen, functies en lijsten.

## 🚀 Functies

* Inkomsten toevoegen
* Uitgaven toevoegen
* Saldo berekenen
* Overzicht van alle inkomsten en uitgaven bekijken
* Gegevens automatisch opslaan
* Gegevens automatisch laden als je het programma opnieuw start

## 🛠️ Gebruikte technieken

* Python 3
* JSON
* Bestandsverwerking
* Lijsten en dictionaries
* `if`, `for` en `while`
* `input()` en `print()`

## ▶️ Programma starten

1. Download of clone deze repository.
2. Zorg dat Python 3 op je computer staat.
3. Open de map in je terminal of command prompt.
4. Voer het volgende commando uit:

```bash
python budget_tracker.py
```
